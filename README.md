# 📚 Collection Manipulator (Student Management System)

A feature-rich CLI (Command Line Interface) Python application designed to manage and manipulate student record collections. Built using Python's modern structural pattern matching (`match-case`) and formatted with ANSI color codes for a vibrant user experience.

---

## 👨‍💻 Developer Information
* **Programmer:** Chirag Kori
* **Language:** Python 3.10+

---

## 🌟 Features

* ➕ **Add Student Details:** Store GR ID, Name, Age, Date of Birth, and Course in structured dictionaries.
* 📋 **View All Records:** Display complete details of all registered students with color-coded formatting.
* 🔍 **Search Records:** Look up student details by their unique **GR ID**.
* ✏️ **Update Student Details:** Flexible sub-menu to update individual fields (Name, Age, DOB, or Course).
* 🗑️ **Delete Specific Record:** Safely remove a student by entering their GR ID.
* 🔐 **Clear All Records:** Password-protected bulk deletion feature (Requires Admin password).
* 🎨 **Interactive UI:** ANSI color codes make terminal logs and messages easy to read and visually appealing.
